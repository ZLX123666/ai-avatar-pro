# AI Avatar Pro · 赚钱操作手册

## 📁 文件结构
```
ai-avatars-pro/
├── index.html    ← 主产品页面（用户看到的面​​）
├── admin.html    ← 订单管理后台（你看的）
├── qr-wechat.png ← 👈 你的微信收款码（需自己放）
├── qr-alipay.png ← 👈 你的支付宝收款码（需自己放）
└── README.md     ← 本文件
```

## 🚀 上线步骤（3 步）

### 第 1 步：放上你的收款码

1. 打开微信 → 我 → 服务 → 收付款 → 二维码收款 → **截图保存**
2. 打开支付宝 → 收钱 → **截图保存**
3. 把两张截图放到这个目录，命名为：
   - `qr-wechat.png`（微信）
   - `qr-alipay.png`（支付宝）
4. 打开 `index.html`，搜索 `qr-placeholder`，把两个占位 div 替换成：
   ```html
   <img src="qr-wechat.png" alt="微信支付">
   ```
   ```html
   <img src="qr-alipay.png" alt="支付宝支付">
   ```

### 第 2 步：部署到线上

**免费方案 — GitHub Pages：**
```bash
# 在 ai-avatars-pro 目录下
git init
git add .
git commit -m "AI Avatar Pro launch"
git remote add origin https://github.com/你的用户名/ai-avatar-pro.git
git push -u origin main
```
然后在仓库 Settings → Pages → 选 main 分支 → Save。几分钟后就能访问。

**国内方案 — 腾讯云静态托管 / 阿里云 OSS：**
把整个文件夹上传即可。

## 📋 订单管理

打开 `admin.html` 即可查看所有订单。

- 所有订单数据存储在浏览器的 localStorage 中
- 用户提交订单后，状态为"待处理"
- 你生成头像并发送后，点击"标记已交付"
- 可导出 CSV 报表
